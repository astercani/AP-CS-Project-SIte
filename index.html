Scratch Game: Dragon Adventure

Link to Play Scratch Game

For this project, we were required to create a video game on Scratch. My partner and I decided to create a runner game loosely inspired by the Google dinosaur game that appears when you have no Wi-Fi. The game's main character is a dragon the player controls to jump over a knight running at you. If you hit a knight you lose two points and if you manage to jump over the knight without being hit you gain two points, once you hit fifty points you win the game and a castle appears.

Snap Project: Draw a Spiral

Link to See Spiral Project

For this project, my partner and I created code in snap that would create a spiral. You can change what the spiral looks like by changing the values of the inputs in the code. The code was fairly simple to write. We had to get the sprite to more a constantly increasing amount of steps so it would spiral outwards. We also had to be able to control the angle that it turned so we could change the shape of the spiral. There weren't a ton of roadblocks in this project. The most challenging part was troubleshooting the code to make sure it ran properly. The best part of this was seeing how many different shapes our code could make after it ran properly.

Snap Game: Planting Native CA Plants

Link to Play Snap Game

The purpose of our program is to teach kids in California about some of the plants native to our state and how to plant them in their garden. This game allows players to click on flowers to learn about them then drag the flowers to gardening boxes. Once the flower is placed in a garden box the player can drag a watering can over the flower to watch it grow if placed in the correct box or wilt if dropped in the wrong one. A player would click on, move and drop sprites which is our input. The output from that would be the chat boxes from the flower sprites telling more about what type of plant they are and the flower sprites changing costumes when dropped in the garden boxes.
